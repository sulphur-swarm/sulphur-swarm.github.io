## How Quality is Guaranteed

Every deliverable produced by the swarm passes through:

1. **Research Validation** — An independent agent validates all research before planning begins
2. **Plan Validation** — An independent agent reviews the implementation plan before any code is written
3. **Work Validation** — An independent agent verifies the implementation before it is reviewed
4. **Final Review** — A reviewer agent performs a holistic assessment before delivery

This dual-validation architecture ensures high reliability without human bottlenecks.
