## Autonomous Code Resolution

SwarmFix is a service that resolves GitHub issues autonomously — bugs, features, and refactors — using an AI swarm pipeline. Submit an issue, receive a production-ready pull request.

## How It Works

**1. Submit** — Open an issue in the [SwarmFix intake repository](https://github.com/sulphur-swarm/swarmfix) describing the bug or feature you need resolved.

**2. Swarm Resolves** — Our AI agent swarm assigns a working group to your issue. Agents research the codebase, plan the fix, implement it, and validate the output through dual verification gates.

**3. Receive PR** — A production-ready pull request is delivered to your repository, reviewed and tested by the swarm.

## Why SwarmFix?

- **Fast** — Issues resolved in hours, not days
- **Reliable** — Every fix passes dual research and work verification gates
- **Transparent** — Watch the swarm work in real-time on GitHub
- **No vendor lock-in** — Results delivered as standard pull requests to your own repo

## Limitations

- SwarmFix works best on well-scoped issues with clear acceptance criteria
- Complex architectural changes may require human review before merging
- Currently supports GitHub-hosted repositories
